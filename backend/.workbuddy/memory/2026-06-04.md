### A同学实现完成（认证+用户+JWT基础设施）

实现文件（4个）：
- `JwtUtil.java`：完整实现 generateToken/getUserIdFromToken/validateToken，基于 jjwt 0.12.6
- `AuthInterceptor.java`：拦截 /api/**（除 /api/auth/login），校验 Bearer Token，解析 userId 注入 request 属性
- `AuthServiceImpl.java`：wxLogin 实现，课设版用 code 作为用户标识（phone 字段），查/创建用户 + member_ext，签发 JWT，组装 LoginVO
- `UserServiceImpl.java`：getProfile（联查 users+member_ext+reservations+user_coupons，含脱敏/等级映射/积分升级计算）、verifyRealname（写入实名信息+身份证脱敏返回）

关键设计决策：
- Users 表无 openid 字段，课设版用 phone 字段存储 code 作为用户标识（注释标明了正式环境的改造方式）
